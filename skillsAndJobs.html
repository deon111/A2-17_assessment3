UI Designer

Brief:

A successful UI designer will have a great eye for design, using the latest techniques and trends to deliver top notch prototypes, ready for development. Having knowledge of Adobe XD and basic front-end coding languages will be paramount for creating these prototypes. Confidence and being able to work well within a team is vital as you will need to interact with team members, as well as presenting designs and prototypes.

    Responsibilities:
  • Collaborating visuals, product direction and experience with engineers and developers.
  • Implementing unique, simple and user-friendly ideas to overcome design issues.
  • Creating Story boards, wireframes, user and process flows to communicate design ideas.
  • Present designs and key prototypes to peers and clientele.
  • Conduct and evaluate user research.
    Required Skills:
  • Creativity with design and problem solving
  • JavaScript, CSS, HTML
  • Proficiency with Adobe XD
  • Visual design skills
  • Bachelor of UI design
  • Minimum one-year UI designer experience
  • Stong portfolio
  • Confident presentation skills
